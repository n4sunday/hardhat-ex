# Hardhat

## Get start

```bash
npm init -y
```

```bash

npm install --save-dev hardhat
```

```bash
npx hardhat
```

```bash
npm install --save-dev @nomiclabs/hardhat-waffle
```

## Compile Contract

```bash
npx hardhat compile
```

## Run Task

```
npx hardhat <task_name>
```

## Run Test

```bash
npx hardhat test
```
